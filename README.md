# Tappa&sup3; (PWA)

Progressive Web App (PWA) for Tappa&sup3; HTML5 Mobile Game


## Development

1. Install node modules with `yarn`
2. Run the local server with `yarn start`
3. Point your browser to `localhost:3000`
